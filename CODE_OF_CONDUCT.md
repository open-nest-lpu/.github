# OpenNest Code of Conduct

## Our Standards

We aim to create a respectful, collaborative environment.

### Expected Behavior

- Be respectful
- Be constructive
- Be inclusive
- Collaborate openly

### Unacceptable Behavior

- Harassment
- Discrimination
- Toxic behavior
- Personal attacks

## OpenNest Culture

- Build over talk
- Learn together
- Respect time
- Help others grow

## Enforcement

Core Council reserves the right to remove harmful contributors.
